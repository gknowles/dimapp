== Overview
Windows framework for server and console applications

* Run in console or as service
* XML config files
* Diagnostics
** Logging to console and/or file
** Performance counters
** Crash dump files
** Web UI
*** External files
*** Compiled into binary
* Task queue
* Timers
* Containers
** bitspan/bitview
** charbuf
** intset
** strtrie
** tempheap
** tokentable - string to/from enum bimap
** file io
*** sync/async
*** monitor for change
*** ruby style filename globbing
** Net
*** tcp
*** http2
*** tls via SSPI
*** windows pipes
* Utilities
** cmtupd
** docgen
** pargen
** resupd
** tnet
** xsdgen
